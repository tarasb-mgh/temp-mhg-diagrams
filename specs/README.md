# Specification Directory

## Numbering Convention

Specs follow sequential numbering: `NNN-short-name`. New specs MUST use the
next available number (check local branches, remote branches, and this
directory).

## Historical Numbering Collisions

Early specs (001, 003) were created before the sequential numbering rule
was enforced. Multiple specs share the same number but are distinct features
distinguished by their short name.

From spec 004 onward, numbering is unique and sequential. The next
available number is determined by scanning all sources (branches, specs
directory) for the highest existing number.
